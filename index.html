Instructor : Asif Ali Banka

Course Description

The course is an introduction to operating system design and implementation. The course will discuss concepts of operating systems as they actually appear in computer systems. We will In'Sha Allah cover the basics of operating system structure, concepts of concurrency, processes, interprocess communication and synchronization. A consistent foundational approach towards scheduling algorithms and need to multiple algorithms will be discussed. Memory management, filesystems and networking will be discussed to encourage holistic approach towards computer system design.

Pre Requiste

Students are expected to be comfortable with problem solving (using any programming language) alongwith solid foundation of Computer Architecture. Though the relevant topics will be discussed as and when required but students are expected to cover the topics for smooth execution of class.
